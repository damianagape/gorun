This module deploys observability stack:

- [OpenTelemetry](https://opentelemetry.io/),
- [Grafana](https://grafana.com/oss/grafana/),
- [ClickHouse](https://clickhouse.com/clickhouse),
- [OpenTelemetry exporter for ClickHouse](https://github.com/open-telemetry/opentelemetry-collector-contrib/tree/main/exporter/clickhouseexporter),
- [Grafana datasource plugin for ClickHouse](https://github.com/grafana/clickhouse-datasource).
